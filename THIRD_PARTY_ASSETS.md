# Third-Party Assets - PrismaUI_F4

## Controller button prompts (Julio Cacko - Free Input Prompts)

The controller button art shipped at `PrismaUI_F4/views/icons/ControllerButtons/` (used by the V9
controller-prompt API and its shared renderer `views/prisma_controller_glyphs.js`) is from
**Julio Cacko - "Free Input Prompts"**, https://juliocacko.itch.io/free-input-prompts, used under its
stated free-to-use terms (free for personal and commercial projects; credit appreciated, not required).

Only the two sets the framework renders are bundled: `XGamepad/Default` (Xbox) and `P5Gamepad/Default`
(PlayStation 5). The full pack (Xbox / PS4 / PS5 / Switch / Keyboard-Mouse, multiple style variants) is
at the itch.io link above.

Credit: **"Input prompt art by Julio Cacko (juliocacko.itch.io/free-input-prompts)."**
